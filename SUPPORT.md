# Support Policy

## Supported platforms

- Windows amd64
- Linux amd64
- Linux arm64
- macOS amd64
- macOS arm64

## Runtime requirements

- `lenv` can use host QEMU or managed runtime fallback.
- Internet access is required for first-time rootfs/runtime downloads.

## Issue reporting

When opening issues, include:

- `lenv version` output (or tag/commit)
- OS + architecture
- `lenv runtime provenance` output
- `lenv provenance` output
- Relevant command and full error text

## Security concerns

Report security issues privately through repository security reporting channels when available.
